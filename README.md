# ENSIA-students-performance
A data mining–based analysis of ENSIA students’ performance, focusing on pattern discovery, student profiling, and performance prediction using supervised and unsupervised learning methods.

## Objectives
- Explore students’ academic data
- Identify performance patterns
- Predict students at risk of failure

## Data
The dataset includes academic scores, attendance, and demographic attributes of ENSIA students.

## Methods
- Descriptive analysis
- Clustering
- Classification

## Tools
Python, Pandas, Scikit-learn, Matplotlib
